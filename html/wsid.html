Uniform Cost Search

Purpose

  • Provide users with a visualization of the uniform cost search algorithm. Module provides users with a graph building tool that can be used to build custom graphs, a set of controls to assist with the graph builder tool, and a view for the explanation of the algorithmic steps as they are performed on the user's custom graph.

Methods

ucs

Function:

  • Allows users to perform graph searches on custom graphs using the uniform cost search algorithm.

Returns:

  • The path information for the shortest path found from start to end, or if no end node was given, a tree of the shortest paths from the start node to every other node in the graph.

Request Format:

  • {"method": "ucs", "nodes": Nodes, "edges": Edges, "start": nodeID, "end": nodeIDString}

Request Parameters:

  • method
    • Value: ucs Name must match exactly for method to function correctly.
  • nodes
    • Value: {"nodeID_1": {"x": nodeX Int , "y": nodeY Int }, . . . , "nodeID_N": {"x": nodeX Int , "y": nodeY Int }}
  • edges
    • Value: {"edgeID_1": {"weight": edgeWeight Float , "parent_1": nodeID String , "parent_2": nodeID String }, . . . , "edgeID_N": {"weight": edgeWeight Float , "parent_1": nodeID String , "parent_2": nodeID String }}
  • start
    • Value: nodeID String
  • end (optional)
    • Value: nodeID String

Response Format:

  • {"steps": Steps, "error": Error}

Response Parameters:

  • steps
    • [{"frontier": [ Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } , . . . , Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } ], "path": Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } , "total_path": [ Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } , . . . , Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } ]}, . . . , {"frontier": [ Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } , . . . , Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } ], "path": Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } , "total_path": [ Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } , . . . , Path Custom data structure of format {"nodes": [ nodeID ,..., nodeID ], "cost": val } ]}]
  • error
    • Value: Error Int

Minimax Adversarial Search

Purpose

  • Allows users to interact with an intelligent agent utilizing an adversarial search algorithm in a game of chess.

Methods

ismovelegal

Function:

  • Used to check the legality of the provided move on the provided board.

Returns:

  • A boolean value indicating the legality of the provided move.

Request Format:

  • {"method": "ismovelegal", "board": Board, "move": Move, "flags": Flags, "en_passant_targets": Targets, "player": Player, "opponent": Opponent}

Request Parameters:

  • method
    • Value: ismovelegal Name must match exactly for method to function correctly.
  • board
    • Value: Board 2D character array with possible values in the set ['R','r','B','b','T','t','Q','q','K','k','P','p','.'], with uppercase indicating white pieces and lowercase indicating black pieces.
  • move
    • Value: Move Custom data structure of format {"from": [ row , col ], "to": [ row , col ]}
  • flags
    • Value: [ white_king Boolean , white_krook Boolean , white_qrook Boolean , black_king Boolean , black_krook Boolean , black_qrook Boolean , en_passant Boolean ]
  • en_passant_targets
    • Value: Targets Custom array of the format [[ row , col ],[ row , col ]]
  • player
    • Value: $classes$char@Player@char$
  • opponent
    • Value: $classes$char@Opponent@char$

Response Format:

  • {"result": Result, "checkmate": Checkmate, "error": Error}

Response Parameters:

  • result
    • Result Boolean
  • checkmate
    • Checkmate Boolean
  • error
    • Error Int
