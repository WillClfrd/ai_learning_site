Modules

Uniform Cost Search

Purpose

  • Provide users with a visualization of the uniform cost search algorithm. Module provides users with a graph building tool that can be used to build custom graphs, a set of controls to assist with the graph builder tool, and a view for the explanation of the algorithmic steps as they are performed on the user's custom graph.

Methods

ucs

Function:

  • Allows users to create custom graphs that they can then perform graph searches on using the uniform cost search algorithm.

Returns:

  • The path information for the shortest path found from start to end, or if no end node was given, a tree of the shortest paths from the start node to every other node in the graph.

Request Format:

  • {"method": "ucs", "nodes": Nodes, "edges": Edges, "start": nodeID, "end": nodeIDString}

Request Parameters:

  • method
    • Value: "ucs"
  • nodes
    • Value: {"nodeID_1": {"x": nodeX, "y": nodeY}, . . . , "nodeID_N": {"x": nodeX, "y": nodeY}}
  • edges
    • Value: {"edgeID_1": {"weight": edgeWeight, "parent_1": "nodeID", "parent_2": "nodeID"}, . . . , "edgeID_N": {"weight": edgeWeight, "parent_1": "nodeID", "parent_2": "nodeID"}}
  • start
    • Value: "nodeID"
  • end (optional)
    • Value: "nodeID"

Response Format:

  • {"steps": Steps, "error": Error}

Response Parameters:

  • steps
    • [{"frontier": [PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}, . . . , PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}], "path": PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}, "total_path": [PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}, . . . , PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}]}, . . . , {"frontier": [PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}, . . . , PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}], "path": PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}, "total_path": [PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}, . . . , PathCustom data structure of format {"nodes": [nodeID,...,nodeID], "cost": val}]}]
  • error
