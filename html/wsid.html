Modules

Uniform Cost Search

Purpose

This module is meant to be used to create graphs and perform a graph search on the created graphs using the uniform cost search algorithm.


Methods

ucs

Function:

  • Contructs a graph using the given information and performs a graph search on the constructed graph.

Returns:

  • Either the path from the start node to the end node or the paths in a minimum spanning tree with the given start node as its root if no end node is passed in the method call.

Request Format:

  • {"method": "ucs", "nodes": Nodes , "edges": Edges , "start": nodeID , "end": nodeID }

Request Parameters:

  • method : "ucs"
  • nodes : structure containing node information following format
  • edges : structure containing edge information following format
  • start : string id for graph search starting node
  • end ( optional ): string id for graph search end node

Response Format:

  • {"steps": StepsArray , "error": ErrorFlag }

Response Parameters:

  • steps : list of dictionaries containing information for the frontier and selected path at each step of the algorithm in format [{"frontier": [,...,], "path": },...,{"frontier": [,...,], "path": }]
  • error : int value indicating error status for method call (0 for success, any other number for error)
