Uniform Cost Search

Purpose

  • The uniform cost search module is meant to be used for demonstrating the steps taken to perform graph searches using the uniform cost search algorithm. The module includes a graph builder to allow users to construct custom graphs to perform the search on, as well as a description panel used to provide text descriptions of each step of the algorithm.
